# Getting Started

## 1. Clone the repository

```bash
git clone your-repo
```

## 2. Open the project

Open:

docethics.html

No build step required.

---

## 3. Edit content

Content is defined in JSON:

/data/cs.json
/data/en.json
/data/ua.json

---

## 4. Modify structure

You can:

- add sections
- add subsections
- edit markup
- add skins

---

## 5. No backend needed

Everything runs:

- client-side
- no database
- no framework

---

## Recommended use cases

- teaching materials
- research structures
- ethical frameworks
- interactive essays
